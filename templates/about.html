Key Features

ML Traffic Prediction
Random Forest classifier with 94%+ accuracy predicts Low / Medium / High traffic levels using time, day, vehicles, weather, and road type.
Live Map Integration
Leaflet.js + OpenStreetMap with Nominatim geocoding. Enter any city name and the map flies to that location with a glowing marker.
Analytics Dashboard
Hourly trend chart, day-wise bar chart, traffic distribution donut, peak-hour analysis, and a full 7×24 traffic heatmap.
Animated Traffic Light
SVG traffic light component with neon glow animations that lights up in the correct color matching the prediction result.
Confidence Scores
Animated probability bars show the model's confidence in each traffic level (Low / Medium / High) with percentage breakdowns.
Render Deployment
Fully configured for one-click Render deployment via render.yaml. Build command auto-trains the model during deployment.

Technology Stack

🐍
Flask
Lightweight Python web framework serving all routes and REST API endpoints. Deployed via Gunicorn on Render.
🌲
scikit-learn
RandomForestClassifier with 200 estimators, class balancing, and 5 engineered features. Serialized with pickle.
🗺️
Leaflet.js
Open-source maps with OpenStreetMap tiles. Nominatim geocoding for city search. Custom animated markers.
📊
Chart.js 4
Line, bar, doughnut, and horizontal bar charts with custom dark theme styling and animated data loading.
🎨
CSS Design System
Custom CSS variables, glassmorphism cards, Orbitron + Rajdhani fonts, animated grid background, neon glow effects.
🚀
Render Cloud
Free-tier hosting with render.yaml. Auto build trains the ML model, Gunicorn serves the Flask app in production.
